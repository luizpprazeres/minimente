## 🎯 Os 5 Padrões de Erro Mais Comuns

### 1. Atrasar tratamento para pedir exame confirmatório
**Em diagnósticos clínicos clássicos ou emergências: TRATE PRIMEIRO.**

| Situação | ❌ Erro Comum | ✅ Correto |
|---------|-------------|---------|
| Herpes Zoster oftálmico | PCR / Tzanck | Antiviral imediato |
| Meningite TB + papiledema | Aguardar PL | RIPE + corticoide |
| HSE (encefalite herpética) | Aguardar PCR do LCR | Aciclovir IV empírico |
| Isquemia arterial de membro | Doppler vascular | Embolectomia cirúrgica |

### 2. Tratar o exame, não o paciente
**Novo achado laboratorial → revise os medicamentos de uso contínuo.**

- Hipercalemia grave → checar iECA, AINEs, espironolactona
- Hipoglicemia recorrente → checar sulfonilureias
- Anticoagulação subterapêutica → checar griseofulvina (indutor CYP450)
- Neuropatia periférica + anemia → checar isoniazida (def. B6)

### 3. Intervenção agressiva em achado benigno/assintomático
**Nem todo achado pede intervenção. O AMC testa o "quando não fazer".**

- Prolapso vaginal leve assintomático → observação
- Cisto renal simples incidental → observação
- Bloqueio AV 1º grau na doença de Lyme → doxiciclina oral (não ceftriaxona IV)

### 4. Ignorar a fisiologia específica do acometimento
**O mesmo diagnóstico em contextos diferentes exige condutas opostas.**

- IAM parede inferior → sempre checar derivações direitas (V4R) para VD
- VD envolvido → pré-carga dependente → fluidos, NUNCA nitratos/diuréticos
- WPW + FA → NUNCA bloqueadores do nodo AV (risco FV letal)

### 5. Confundir timing: crise aguda vs. manutenção crônica
**A questão diz "agora" ou "longo prazo"? Isso muda tudo.**

| Condição | Crise Aguda | Manutenção |
|---------|------------|-----------|
| FA | Cardioversão / controle FC | Dabigatrana |
| Gota | AINEs / colchicina | Alopurinol (após crise) |
| AR | Corticoide / AINE | Metotrexato |
| DM2 hospitalizado | Insulina SC | Retomar metformina após alta |

## 📚 5 Dicas de Estudo para Dominar o AMC

### Dica 1: Estabilize antes de investigar
> Quando o paciente está instável ou o diagnóstico é clínico clássico, a resposta é **agir**, não pedir exame.

- Meningite bacteriana suspeita → antibiótico antes da TC
- Meningite TB com hipertensão intracraniana → RIPE + corticoide antes da PL
- Hiponatremia com convulsão → SF 3% antes de qualquer outro passo

### Dica 2: A lista de medicamentos é um personagem da questão
> Sempre que um paciente desenvolve algo novo, a primeira pergunta é: *"Algum medicamento poderia estar causando isso?"*

- iECA → hipercalemia, tosse, angioedema
- Diuréticos tiazídicos → hiperuricemia (Gota), hiponatremia
- Griseofulvina → reduz efeito da varfarina
- Isoniazida → neuropatia periférica (def. B6)
- Heparina → HIT (contagem de plaquetas entre dias 5–10)

### Dica 3: Domine gasometria + ECG com conduta
> Não basta interpretar o exame. A questão quer a **ação clínica**.

| Gasometria | Diagnóstico | Conduta |
|-----------|------------|---------|
| pH↓, PCO₂↑ (acidose resp.) grave em DPOC | Exacerbação grave | Intubação + VM |
| pH↑, PCO₂↓ (alcalose resp.) aguda | TEP | Anticoagulação + angioTC |
| pH↓, HCO₃↓, gap normal | Diarreia / IRA | Tratar causa base |

| ECG | Diagnóstico | Conduta |
|-----|------------|---------|
| Supra V1-V4 | IAM anterior (LAD) | Reperfusão urgente |
| Supra II, III, aVF + V4R | IAM inferior + VD | Fluidos (não nitratos!) |
| Ritmo irregular, sem P | FA | Estável: FC control / Instável: cardioversão |
| Delta wave + taquicardia | WPW | Procainamida (NUNCA verapamil/adenosina) |

### Dica 4: Red flags mudam o diagnóstico diferencial
> Um detalhe na história pode transformar o diagnóstico mais óbvio no errado.

- Pupila dilatada + paralisia NC III → aneurisma (cirúrgico)
- Pupila preservada + paralisia NC III → isquemia diabética (conservador)
- AVC + dispositivo implantado → TC, não RM
- Hipercalemia + uso de heparina → HIT (não ajustar dieta)

### Dica 5: Diferencie estágio agudo vs. crônico sempre
> Leia a questão perguntando: *"O paciente está em crise agora ou entre as crises?"*

- AR **em crise** → AINEs / corticoide intra-articular
- AR **fora de crise** → Metotrexato (DMARD)
- FA **instável** → Cardioversão
- FA **estável, longo prazo** → Anticoagulação oral
- Gota **em crise** → Anti-inflamatórios
- Gota **controlada** → Alopurinol para prevenção
